# Remote++

增强 NVDA 远程访问功能，为高级用户提供更高效的操作体验。

## 功能

### 连接管理器

快速保存、整理和连接常用的远程会话。

* 使用分组整理连接
* 按名称或主机搜索连接
* 从列表一键连接
* 复制连接链接到剪贴板

### 交换控制模式

在控制另一台计算机和被控制之间即时切换，无需断开连接。

### 连接到默认服务器

一键连接到已配置的自动连接服务器。

## 连接管理器

连接管理器提供了一个便捷的界面来管理您的远程连接。

### 列表快捷键

| 操作 | 快捷键 |
|------|--------|
| 以反向模式连接 | `Shift+回车` |
| 上移连接 | `Alt+上光标` |
| 下移连接 | `Alt+下光标` |
| 编辑连接 | `F2` |
| 删除连接 | `Delete` |
| 全选 | `Ctrl+A` |

### 上下文菜单

右键单击连接可访问更多选项：

* 连接 / 以反向模式连接
* 编辑
* 复制链接
* 设置为自动连接
* 上移 / 下移
* 删除


## 快捷键

| 命令 | 手势 |
|------|------|
| 打开连接管理器 | `NVDA+Control+Shift+M` |
| 交换控制模式 | `NVDA+Control+Shift+S` |
| 连接到默认服务器 | 无 |

## 菜单项

所有功能也可通过 NVDA 远程访问菜单使用:

* 连接管理器...
* 交换控制模式
* 连接到默认服务器（仅在已配置自动连接时显示）

## 要求

* NVDA 2025.1 或更高版本，需启用内置远程访问功能

## 安全性
为最大限度确保安全并防止意外访问，该插件在安全桌面（如 UAC 提示、Windows 登录界面）上将被禁用。
